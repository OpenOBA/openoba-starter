# OpenOBA 贡献者许可协议（CLA）

> **完整 CLA 法律文本请参阅**：[docs/open-source/policies/CLA.md](./docs/open-source/policies/CLA.md)

## 概要

向 OpenOBA 项目提交 Pull Request 即表示你同意以下条款：

1. **著作权转让**：贡献的著作财产权转让给深圳市秒镜科技有限公司（署名权保留）
2. **专利许可**：贡献涉及的专利以非独占、永久、免费方式许可
3. **原创性担保**：你保证贡献为原创，不侵犯第三方权益
4. **双重许可**：公司可以对贡献进行 MIT/BSL 双重许可分发
5. **撤回机制**：可书面通知 90 日后撤回，不影响已分发版本

**提交 PR = 电子签署 CLA**，符合《中华人民共和国电子签名法》第十三条。

完整法律文本（11 条，中英对照）请参阅上方链接。
