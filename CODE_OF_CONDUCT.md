# OpenOBA 社区行为准则

> **完整行为准则请参阅**：[docs/open-source/policies/CODE_OF_CONDUCT.md](./docs/open-source/policies/CODE_OF_CONDUCT.md)

本项目遵循 [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) 改编的行为准则。

我们承诺为所有参与者提供一个无骚扰的社区环境，不论年龄、体型、残疾、族裔、性别特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、容貌、种族、宗教信仰、性别取向。
