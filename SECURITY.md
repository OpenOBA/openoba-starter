# OpenOBA 安全策略

> **完整安全策略请参阅**：[docs/open-source/policies/SECURITY.md](./docs/open-source/policies/SECURITY.md)

本项目的安全策略包含：

- **报告安全漏洞**：请发送加密邮件至 **postmaster@openoba.com**
- **SLA 承诺**：24 小时内确认，7 个工作日内初步评估
- **严重性分级**：CVSS 3.1（Critical/High/Medium/Low）
- **Safe Harbor 声明**：善意安全研究人员不面临法律诉讼
- **CVE 编号机制**：通过 GitHub Security Advisories (GHSA) 分配

**请不要在公开 Issue / Discussion / PR 中报告安全漏洞。**
