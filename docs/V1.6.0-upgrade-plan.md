# V1.6.0 升级计划

> 制定人：唐浩然
> 基线：V1.5.1
> 日期：2026-06-28

## 目标

增强 Agent 执行能力闭环：分析需求 → 修改代码 → 编译验证 → 测试 → 提交记录 → 前端可见。

## 升级内容

### 1. Agent 上下文增强

- 元镜引擎扫描结果注入 Agent System Prompt（实体结构、API 路由、编码规范）
- Skill 激活时自动注入精准业务上下文
- 质量门禁规则纳入 Agent 行为约束

### 2. 工具链完善

- Agent 可自主执行：代码编辑 → 编译检查 → 测试 → Git 提交
- ERDL 支持运行时字段注册，新增 Entity 字段后 Agent 可立即操作
- Agent 任务进度可视化 + 长任务不被中途清理

### 3. 安全与体验

- Agent 内部执行过程不泄露到前端渲染层
- Agent 身份激活机制修复

## 验收标准

- Agent 收到"给商品表新增字段"指令后：分析 → 改 Entity → 编译通过 → 测试通过 → 自动 commit
- Agent System Prompt 包含元镜生成的实体/API/规范上下文
- 长任务（1h+）不会因超时中断，前端能感知 Agent 仍在执行
- ERDL 新字段无需重启即可被 Agent 操作
