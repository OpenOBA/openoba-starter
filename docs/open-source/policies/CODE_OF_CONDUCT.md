# OpenOBA 行为准则

> 版本：V1.0 · 生效日期：2026-06-15

---

本项目遵循 [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)。

## 我们的承诺

为了营造一个开放和友善的环境，我们作为贡献者和维护者承诺：无论年龄、体型、残障、种族、性别认同和表达、经验水平、国籍、个人外貌、宗教、性取向如何，参与本项目不会受到骚扰。

## 我们的标准

有助于创造积极环境的行为包括：

- 使用友善和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表现出同理心

不可接受的行为包括：

- 使用带有性暗示的语言或图像，以及不受欢迎的性关注或挑逗
- 挑衅、侮辱/贬损评论、人身攻击或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息（如地址）
- 在专业环境中可被合理认为不适当的其他行为

## 执行

维护者有责任澄清可接受行为的标准，并应对任何不可接受的行为采取适当和公正的纠正措施。

维护者有权删除、编辑或拒绝不符合本行为准则的评论、提交、代码、问题和其他贡献，或暂时/永久禁止任何他们认为不适当、威胁、冒犯或有害的贡献者。

## 范围

本行为准则适用于所有项目空间，也适用于个人在公共场合代表项目或社区时。

## 报告

侮辱、骚扰或其他不可接受的行为可向项目团队报告至 **postmaster@openoba.com**。所有投诉将被审查和调查，并将视情况采取必要和适当的措施。项目团队有义务为事件报告者保密。
