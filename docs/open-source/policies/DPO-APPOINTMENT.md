# OpenOBA DPO 任命书

> 版本：V1.0 · 生效日期：2026-06-15

---

深圳市秒镜科技有限公司（以下简称"公司"）根据《个人信息保护法》等相关法律法规的要求，正式任命：

**唐启鑫** 担任公司 **数据保护官（DPO）**。

---

## DPO 职责

1. 监督公司数据处理活动，确保符合适用法律法规
2. 作为数据主体与监管机构的联系窗口
3. 开展数据保护影响评估（DPIA）
4. 维护数据处理活动记录
5. 向管理层提供数据保护建议
