# OpenOBA 隐私政策

> 版本：V1.0 · 生效日期：2026-06-15 · 最后更新：2026-06-15

---

## 概述

深圳市秒镜科技有限公司（以下简称"我们"）运营 OpenOBA 平台。本隐私政策说明我们如何收集、使用和保护您的信息。

---

## 我们收集的信息

### 您主动提供的信息
- 账户信息（用户名、邮箱）
- LLM API Key（AES-256-GCM 加密存储）

### 自动收集的信息
- 基本使用统计（功能使用频率）
- 错误日志（用于故障排除）

### 我们不收集的信息
- 您的业务数据（ERDL 规则、客户数据等存储在您的本地实例中）
- 聊天/对话内容
- 个人身份信息（除必要账户信息外）

---

## 数据存储

- OpenOBA 是**本地部署**软件，您的业务数据存储在您自己的服务器上
- LLM API Key 使用 AES-256-GCM 加密存储在您本地数据库中
- 我们不访问、不传输、不存储您的业务数据

---

## 数据保护

- 传输：所有 API 通信使用 HTTPS
- 存储：LLM Key AES-256-GCM 加密
- 访问：基于角色的访问控制（RBAC）

---

## 您的权利

- 访问您的账户信息
- 更正不准确的信息
- 删除您的账户
- 导出您的数据
